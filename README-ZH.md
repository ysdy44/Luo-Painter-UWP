## 功能说明

洛画 是一款免费开源的绘画工具，仅支持 Windows OS 平台。它针对压感笔进行设计，支持常规，液化，模糊等笔刷，带有丰富的笔刷预设，支持 30 多种滤镜，可以允许用户自由变换、选取需要的像素。如果你需要画画，不妨试试。
